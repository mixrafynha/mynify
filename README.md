# ✨ Overview

Mynify is a modern ecommerce web application focused on performance, scalability and clean user experience.

Built with a modern full-stack architecture using Next.js, TypeScript and Supabase.

The project demonstrates:
- Modern frontend development
- Full-stack architecture
- Authentication systems
- Database integration
- Scalable project structure
- Responsive design principles

# ⚙️ Getting Started

## Clone the repository

```bash
git clone https://github.com/mixrafynha/mynify.git
```

## Navigate into the project

```bash
cd mynify
```

## Install dependencies

```bash
npm install
```

## Start development server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

# Ryfio

Professional print-on-demand ecommerce platform built with Next.js, TypeScript, Tailwind CSS and Supabase.

## Overview

Ryfio helps users design custom products, launch a brand and sell worldwide with print-on-demand fulfillment.

Official domain:

```text
https://ryfio.com
```

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Supabase
- Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Production

```bash
npm run build
npm run start
```

# 🚀 Overview

Ryfio is a **full-stack Print-on-Demand SaaS platform** designed for creators, entrepreneurs and brands who want to build, customize and sell products online without inventory.

The platform combines a modern product editor, cloud infrastructure and an optimized ecommerce workflow into a single scalable application.

Built entirely from scratch, Ryfio focuses on **performance**, **developer experience**, **responsive UI** and **clean architecture**.

# 💻 Local Development

Clone the repository

```bash
git clone https://github.com/mixrafynha/ryfio.git
```

Open the project

```bash
cd ryfio
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open

```text
http://localhost:3000
```

---

# 🚀 Production

```bash
npm run build

npm run start
```
